Überflutungsrisikozonen Deutschland – Zusammenfassung

 

Inhalt: Feature Service & Web Map mit Überflutungsrisikozonen in Überflutungsrisikozonen in Nordrhein-Westfalen Deutschland.

Verwaltung: Daten werden von der Bundesanstalt für Gewässerkunde (BfG) im Auftrag der Wasserwirtschaftsverwaltungen gepflegt und über das nationale Berichtsportal WasserBLIcK bereitgestellt.
Datenbasis: Nationale, flächendeckend homogenisierte Daten, abgestimmt mit der Länderarbeitsgemeinschaft Wasser (LAWA).
Quelle: Hochwasserrisikomanagement-Richtlinie 2. Zyklus (2016–2021), BfG-Metadatenkatalog, zuletzt abgerufen am 27.10.2025.
Verarbeitung:
Download als gezippte File-Geodatabase
Import in ArcGIS Pro
Projektion ins WebMercator WGS84 (3857) Koordinatensystem
Hosting als Feature Service auf ArcGIS Online
Datenstand: Letzte Bearbeitung 2024
